### Overview
EchoMind is a privacy-first voice journal where you record voice notes, and an AI (running locally or self-hosted) extracts emotions, topics, and insights, storing them as searchable embeddings. The model re-trains weekly based on your feedback, becoming uniquely yours.
Learning Goals

```bash
Rust: Data ingestion, API serving, ONNX inference.
ML: Embeddings, fine-tuning, active learning.
PostgreSQL: Vector search with pgvector.
CI/CD for ML: Model registry, retraining pipelines.
Privacy: Local-first AI with user-controlled data.
```
